class LoopStructural.modelling.fault.fault_builder.FaultBuilder(interpolator=None, interpolators=None, model=None, fault_bounding_box_buffer=0.2, **kwargs)¶

Bases: LoopStructural.modelling.features.structural_frame_builder.StructuralFrameBuilder

A specialised structural frame builder for building a fault

Parameters

  • interpolator (GeologicalInterpolator, optional) – the interpolator to use for building the fault frame, by default None

  • interpolators ([GeologicalInterpolator, GeologicalInterpolator, GeologicalInterpolator], optional) – a list of interpolators to use for building the fault frame, by default None

  • model (GeologicalModel) – reference to the model containing the fault

  • fault_bounding_box_buffer (float, default 0.2) – the maximum area around the model domain that a fault is modelled. For high displacement faults this may need to be large, smaller values will be result in fewer degrees of freedom = quicker interpolation
